HyprlandTokyo Night

Desktop ideas to revisit

Notes from the Hyprland setup discussion. The first clean batch was applied separately: smart gaps, empty workspace binds, hypridle starts with Hyprland, and DPMS turns displays off after 10 minutes without automatic lock or suspend.

Design direction

Chosen direction: restrained wallpaper-accented Tokyo Night. Keep the stable dark base for programming, then let shell surfaces use a limited accent. Do not recolor editor, terminal palette, browser content, GTK, Qt, Discord, or other work apps from every wallpaper.

HyDE/Wallbash lesson: full Wallbash is impressive because it propagates wallpaper-derived colors into Hyprland borders, bars, launchers, GTK/Qt, Hyprlock, VS Code, and other templates. For a daily programming desktop, use the idea sparingly.

Ideas not implemented yet

Keybind labels and help menu

  • Convert core binds to bindd so labels appear in hyprctl binds -j.
  • Add Super+/ to open a searchable keybind cheat sheet.
  • Use Wofi at first; consider rofi-wayland, fuzzel, tofi, or anyrun later if richer menus are worth it.

Master layout extras

  • You already have Super+Shift+Return for swapwithmaster and Alt-Tab cycling.
  • Possible later binds: focusmaster, mfact +/-, and orientationcycle.

Special workspaces

  • Special workspaces are scratchpads that pop in and out with a keybind.
  • You already have a dropdown terminal/scratchpad pattern.
  • Possible later named scratchpads: notes, logs, music, AI. Skip unless they beat tmux.

Skipped because tmux already wins

  • Hyprland groups duplicate tmux panes/windows and lose remote laptop continuity.
  • Keep terminal clustering inside tmux unless there is a clear GUI-only use case.

Fullscreen and snapping

  • Leave fullscreen behavior alone for now.
  • Floating-window snapping could help little utility windows like Thunar, file-roller, YAD dialogs, file pickers, audio mixers, calculators, and color pickers.

hyprsunset

  • Potentially useful if you want Night Light/f.lux-style warmth again.
  • Prefer hyprsunset over shader-based approaches because it is not captured in screenshots or recordings.

Visual polish backlog

  • Make Waybar calmer: fewer loud pills, tighter rhythm, clearer active workspace, better urgent states.
  • Restyle Wofi so it feels like part of the same shell, not a default launcher.
  • Refine Dunst notifications with a softer Tokyo Night surface and meaningful urgency colors.
  • Make Hyprlock match the shell: darker blur, cleaner type, one accent ring or field.
  • Reduce bright cyan shadow intensity. Keep cyan as identity, but make the shadow less gamer/rice.
